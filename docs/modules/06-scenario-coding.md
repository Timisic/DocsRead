# 场景二：远程代码编写

通过 Discord 远程控制电脑，完成简单的代码编写任务。

---

## 适合的任务类型

简单、明确、不需要复杂调试的任务最适合：
- 简单工具脚本
- Bug 修复
- 文档生成
- 小型功能开发

---

## 配置 Coding Skills

```bash
"请查阅 /docs/skills/ 下关于 Coding 的文档，帮我配置 Coding Agent，让我可以通过 Discord 远程写代码。"
```

---

## 使用方式

调用 Coding Skill + 描述需求开始写代码。

---

## 案例：Obsidian 笔记整理

配置 Obsidian Skill 后，通过 Discord 保存网页笔记：

```bash
"请帮我：
1. 读取这个链接的内容 https://example.com/article
2. 总结成简短的笔记
3. 保存到我的 Obsidian vault 的 '收藏' 目录
4. 以文章标题作为文件名
5. 添加标签：#收藏 #待读"
```
