# 场景一：自动化论文推送

通过现成的 Skill，自动获取你研究领域的最新论文，推送到 Discord。

---

## 寻找 Skill

在 ClawdHub 搜索 `paper` 或 `arxiv`，或搜索 GitHub 上的 `openclaw-skill arxiv`，找到合适的论文推送 Skill。

---

## 安装与配置

```bash
"请帮我找 ClawdHub 上关于 arXiv 论文推送的 Skill，推荐几个给我"
```

```bash
"请帮我安装刚才推荐的那个论文推送 Skill"
```

```bash
"请帮我配置这个 Skill：
- 论文分类：cs.AI
- 筛选关键词：transformer, diffusion, attention
- 推送时间：北京时间早上 8 点
- 推送到 Discord"
```

---

## 使用效果

测试成功后，后续提及就可以执行论文推送，或是每日自动推送。

```bash
"帮我测试一下这个论文推送功能，现在手动执行一次"
```
